source · 
pub enum ID3D12DeviceVersion {
    Device(ID3D12Device),
    Device10(ID3D12Device10),
}

Variants§

§

Device(ID3D12Device)

Basic device compatible with legacy barriers only, i.e. can only be used in conjunction with ResourceStateOrBarrierLayout::ResourceState.

§

Device10(ID3D12Device10)

Required for enhanced barrier support, i.e. when using ResourceStateOrBarrierLayout::BarrierLayout.
